# 📝 Metin Kontrol — Türkçe Yazım ve Noktalama Denetleyici

Yapay zeka destekli Türkçe yazım, noktalama ve dilbilgisi denetleyici.  
Google Gemini API ile güçlendirilmiştir.

## 🚀 Özellikler

- **🤖 Yapay Zeka Analizi**: Google Gemini API ile derin metin analizi
- **📚 Yerel Sözlük**: 60+ yaygın Türkçe yazım hatası tespiti
- **📌 Noktalama Kontrolü**: 8 farklı noktalama kuralı
- **📝 Dilbilgisi**: Tekrarlanan kelime ve cümle yapısı kontrolü
- **🎨 Renkli Önizleme**: Hatalı kısımlar renkli olarak vurgulanır
- **✅ Otomatik Düzeltme**: Yapay zeka ile düzeltilmiş metin önerisi

## 🛠️ Kullanılan Teknolojiler

| Teknoloji | Kullanım Amacı |
|-----------|----------------|
| Python | Backend programlama dili |
| Streamlit | Web arayüzü framework'ü |
| Google Gemini API | Yapay zeka ile metin analizi |
| Regex | Noktalama ve yazım kalıpları |
| Levenshtein Algoritması | Kelime benzerliği |

## 📦 Kurulum

```bash
pip install -r requirements.txt
```

## ▶️ Çalıştırma

```bash
streamlit run app.py
```

## 🔑 Gemini API Anahtarı

1. [Google AI Studio](https://aistudio.google.com/apikey) adresine gidin
2. Ücretsiz API anahtarı oluşturun
3. Uygulamanın sol menüsünden anahtarı girin
